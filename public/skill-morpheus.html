/morpheus
Research

Speculative dreaming — background thinking, pre-computation, cross-repo patterns, prediction

Description

Enter a deep speculative thinking mode. Morpheus scans across repos, sessions, and learnings to surface non-obvious patterns, pre-compute likely next steps, and generate predictions about project trajectories. Named after the god of dreams — thinking that happens while you sleep, ready when you wake.

Usage

/morpheus /morpheus --deep

Modes

Default

Scan recent sessions + memory, surface top 3 speculative insights

--deep

Full cross-repo pattern scan, long-range predictions, 10+ insights

--focus <topic>

Direct speculation at a specific project or question

--save

Persist insights to ψ/memory/learnings/ for future sessions

วิธีทำงาน

1

Read ψ/memory/ — recent learnings, retrospectives, resonance

2

Scan git history across repos for pattern signals

3

Cross-reference patterns — find hidden connections

4

Generate speculative predictions ranked by confidence

5

Present insights with evidence chain, save if --save

Output

Speculative insight report: patterns found, predictions with confidence %, evidence references, suggested next actions

ใช้เมื่อไหร่?

ต้องการ fresh perspective, stuck on direction, วางแผน roadmap, หรือต้องการ surface patterns ที่ไม่เคยสังเกต

ใช้เมื่อไหร่ไม่ได้?

ไม่ใช่ tool สำหรับ factual lookups หรืองานที่มี spec ชัดเจนแล้ว — morpheus เป็น speculative โดยธรรมชาติ ให้ human verify ก่อน act
