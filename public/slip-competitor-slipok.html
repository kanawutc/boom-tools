04 — Technical Details

§4Technical Details

API spec, SDK, limitations และ integration capabilities

Authentication

  • Branch ID + API Key (ไม่ใช่ Bearer token)
  • Key-per-branch model — แต่ละร้านมี key ของตัวเอง
  • Check quota route: GET /quota

API Response Fields

  • bankCode — รหัสธนาคาร
  • bankName — ชื่อธนาคาร
  • Transaction date, time, amount
  • Sender/Receiver account info
  • Error 1010 with delay handling

Verification Modes

  • ✓ QR Code verification
  • ✓ Slip image (screenshot) verification
  • ✓ รองรับทุกธนาคารไทย
  • ✓ TrueMoney Wallet

SDK & Developer Tools

  • Community SDK: TypeScript (PrakritManStudio/slipok-sdk)
  • SDK timeout default: 10,000ms
  • Docs: slipok.com/api-documentation/
  • ✗ ไม่มี official SDK จาก SlipOK

Integration Limitations

  • ✗ ไม่มี Webhook / Event-driven
  • ✗ ไม่มี Sandbox / Test mode
  • ✗ ไม่มี Batch processing (ยกเว้น 3 สลิปผ่าน Page365)
  • ✗ ไม่มี Shopify integration
  • ✗ ไม่มี Shopee integration

Reliability Notes

  • เว็บไซต์ return 403 บ่อยครั้ง (Cloudflare blocking)
  • Error code 1010 — มี delay handling
  • ไม่มี SLA เปิดเผยต่อสาธารณะ
  • ไม่มี status page สาธารณะ

06–07 — SWOT

§6–7Strengths & Weaknesses

การวิเคราะห์จุดแข็งและจุดอ่อนของ SlipOK เทียบกับตลาด

✓ Strengths

  • 💪 Brand แข็งแกร่งที่สุดในกลุ่ม DR 46 — สูงสุดใน dedicated slip verify
  • 📱 AIS Partnership — ฟรี 3 เดือนสำหรับลูกค้า AIS SMEUNIQUE
  • 🛒 Page365 Integration — อัปโหลด 3 สลิปพร้อมกัน auto-update invoiceUNIQUE
  • 🆓 Free tier ดี — 100 slips/เดือนทุกเดือน สร้างได้ 2 ร้าน
  • 🏢 ที่ตั้งกรุงเทพฯ สีลม RCK Tower — ชั้น 33 สำนักงาน prestige
  • ⏳ อายุกิจการมากที่สุดในกลุ่ม — 3 ปี 7 เดือน ความน่าเชื่อถือสูง
  • 📦 Community TypeScript SDK — มี third-party SDK แล้ว

✗ Weaknesses

  • 💸 ราคาแพงกว่า Thunder ที่ทุก tier — cost/slip สูงกว่าทุกระดับ
  • 🔔 ไม่มี Webhook / Event-driven — ต้อง poll แทน รองรับ real-time ไม่ได้
  • 🧪 ไม่มี Sandbox / Test mode — นักพัฒนาต้องใช้ production ตลอด
  • 📚 ไม่มี Official SDK — community SDK เท่านั้น ไม่มีการ maintain จาก SlipOK
  • 🛍️ ไม่มี Shopify / Shopee integration — พลาด marketplace ใหญ่
  • 🚫 เว็บ return 403 บ่อย (Cloudflare blocking) — UX นักพัฒนาแย่
08 — Market Position

§8Market Position

ตำแหน่งของ SlipOK ในตลาด slip verification ของไทย

ผู้นำตลาดด้าน Brand Awareness

SlipOK ถือเป็น incumbent ที่แข็งแกร่งที่สุดในตลาด dedicated slip verification ด้วย DR 46 ที่สูงสุดในกลุ่ม อายุกิจการมากกว่า 3 ปี 7 เดือน และ partnership พิเศษกับ AIS ที่คู่แข่งรายอื่นไม่มี จุดแข็งหลักของ SlipOK อยู่ที่ ecosystem — การรวม Page365 + LINE Bot + WooCommerce + AIS สร้าง stickiness สูงสำหรับ SME ที่ใช้แพลตฟอร์มเหล่านี้อยู่แล้ว

อย่างไรก็ตาม ในด้าน developer experience และ pricing นั้น SlipOK ยังตามหลัง — ขาด webhook, sandbox, และ official SDK ที่นักพัฒนาสมัยใหม่คาดหวัง ในขณะที่ราคาแพงกว่าคู่แข่งอย่าง Thunder ทุก tier
