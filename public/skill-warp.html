/warp
Navigation

Teleport to a remote oracle node via SSH+tmux — interactive session

Description

Instantly jump into a tmux session on a remote oracle node over SSH. Warp resolves the node alias from the fleet registry, opens an SSH tunnel, attaches or creates a tmux session named after the oracle, and drops you into a live interactive shell. Feels like teleportation across the Oracle network.

Usage

/warp <node> /warp pimpims-imac

Modes

Default

SSH into node, attach existing tmux session or create new one

/warp --list

List all registered warp destinations from fleet registry

/warp --check <node>

Test SSH reachability without entering session

/warp --detach <node>

Run a command on remote node without interactive session

วิธีทำงาน

1

Resolve node alias from fleet registry (maw-js)

2

Test SSH connectivity to target host

3

Open SSH connection with tmux attach or new-session

4

Hand off control — interactive session begins

Output

Live interactive tmux session on remote oracle node. Session persists after disconnect.

ใช้เมื่อไหร่?

ต้องการเข้า Oracle บนเครื่องอื่น (iMac พี่ปิม, server), debug ปัญหาบน remote node, หรือ handoff งานไปยัง oracle อื่น

ใช้เมื่อไหร่ไม่ได้?

ถ้าแค่ต้องการ query ข้อมูลจาก oracle อื่นโดยไม่ต้องเข้า session ใช้ /wormhole แทน — warp เปิด interactive shell ใช้ resources มากกว่า
