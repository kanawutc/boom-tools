/skills-list
Utility

List all Oracle skills with profile tier, type, and script status

Description

Generate a complete, structured inventory of all skills installed in the current oracle. Shows skill name, version, profile tier (core/specialist/community), type (navigation/communication/research/utility), script status (backed by script or pure prompt), and last used date. Essential for understanding oracle capability surface at a glance.

Usage

/skills-list /skills-list --verbose

Modes

Default

Compact table: name, tier, type, script status

--verbose

Full detail: version, description, trigger keywords, last used

--filter <tier>

Show only core / specialist / community skills

--missing

Show skills available in arra-oracle-v3 but not installed locally

วิธีทำงาน

1

Scan .claude/skills/ directory for installed skills

2

Read each skill's metadata (tier, type, version)

3

Check for backing scripts in scripts/ directory

4

Cross-reference with profile tier definitions

5

Render table sorted by tier then alphabetically

Output

Skills inventory table: name, tier, category, script (yes/no), version, description summary

ใช้เมื่อไหร่?

ต้องการรู้ว่า oracle มี skill อะไรบ้าง, onboard oracle ใหม่, audit skill coverage, หรือ plan skill installation

ใช้เมื่อไหร่ไม่ได้?

ไม่ใช่ skill runner — /skills-list แค่แสดง inventory ถ้าต้องการ install skills ใหม่ ใช้ /oracle หรือ /bampenpien
