Competitive Analysis

Strengths & Weaknesses

Strengths

  • Official TypeScript SDK (rdcw-slip npm) — developer-friendly มาก
  • ส่วนหนึ่งของ ecosystem ขนาดใหญ่ (RDCW/Altsera/Strity/EasyDonate)
  • DR 57 สูงสุดในกลุ่มทั้งหมด (จาก parent domain authority)
  • API-first design — clean, structured, testable
  • IP whitelist security — enterprise-grade
  • Status page สาธารณะ (status.rdcw.co.th)

Weaknesses

  • API เท่านั้น — ไม่มี LINE bot, chatbot หรือช่องทาง non-technical
  • ไม่มี pricing transparency สาธารณะ
  • Traffic น้อยมาก (28 visits/month)
  • Target developer เท่านั้น — ตลาดแคบ
  • ผูกกับ RDCW ecosystem — ลูกค้าอื่นอาจไม่เหมาะ
🏗

Market Position

Platform integrator — เสริม slip verify ให้ ecosystem ตัวเอง (RDCW webshop) ไม่ได้ compete ตลาด B2C อย่างจริงจัง มี DR สูงที่สุดเพราะ parent domain แข็งแกร่ง แต่ไม่น่าจะขยายตลาดออกนอก ecosystem ของตัวเอง TypeScript SDK เป็นจุดแข็งสำหรับ developer ที่ต้องการ official library
