PART 1: Requirement Document

สรุปโปรแกรมที่ต้องการ

ชื่อ: AI วิเคราะห์ศักยภาพที่ดิน (Land Feasibility AI Analyzer)

วัตถุประสงค์: วิเคราะห์ที่ดินที่มีลักษณะซับซ้อนได้ใน "ไม่กี่นาที" แทนที่จะใช้เวลา "หลายวันหรือสัปดาห์" ด้วยความแม่นยำ 80-90%

อ้างอิง: คอร์สของ พุฒวดี พรสินธ์ (พี่พีท) — สอนวันที่ 30 พ.ค. 2568 ราคา ฿9,900-13,000

อ้างอิง: Google Form คอร์สต้นฉบับ

Features ที่ต้องมี (จากคอร์ส)

#FeatureรายละเอียดPriority
F1ตรวจสอบสีผังเมืองใส่พิกัด/ที่อยู่ → แสดงสีผังเมือง + ข้อกำหนดการใช้ที่ดิน (กทม. ร่างใหม่ 2568 + ทั่วประเทศ)MUST
F2FAR / BCR / OSR Calculatorคำนวณ Floor Area Ratio, Building Coverage Ratio, Open Space Ratio จากสีผังเมือง + ขนาดที่ดินMUST
F3กฎหมายควบคุมอาคารแสดง setback, ความสูง, ระยะถอยร่น ตาม พ.ร.บ.ควบคุมอาคาร 2522 + กฎกระทรวง + ข้อบัญญัติท้องถิ่นMUST
F4Highest and Best Use Analysisวิเคราะห์ว่าที่ดินนี้ควรพัฒนาเป็นอะไร (คอนโด/บ้านจัดสรร/อาคารพาณิชย์/โรงแรม ฯลฯ) ตามกฎหมาย + ตลาดMUST
F5Feasibility P&L Calculatorคำนวณกำไร-ขาดทุน: ต้นทุนที่ดิน + ก่อสร้าง + ดอกเบี้ย → รายได้จากขาย/เช่า → กำไรสุทธิ + IRR + ROIMUST
F6AI Prompt Engineใช้ AI (Gemini/Claude/GPT) วิเคราะห์ข้อมูลกฎหมายหลายร้อยหน้า → สรุปเป็นคำแนะนำMUST
F7แผนที่ + Overlay ผังเมืองแสดงแผนที่จริง + overlay สีผังเมืองจาก DPT GeoJSON (เหมือน gas station zone checker)SHOULD
F8เปรียบเทียบ Scenarioเปรียบเทียบ 2-3 scenario (เช่น สร้างคอนโด vs บ้านจัดสรร vs ขายที่เปล่า)SHOULD
F9Export PDF Reportสร้างรายงาน PDF สำเร็จรูปสำหรับส่งลูกค้า/นักลงทุนNICE
F10ฐานข้อมูลราคาตลาดราคาประเมินกรมที่ดิน + ราคาตลาดจริงในพื้นที่ใกล้เคียงNICE

แหล่งข้อมูลที่ต้องใช้

กฎหมายและผังเมือง

  • ผังเมืองรวม กทม. (ฉบับ 2556 + ร่างใหม่ 2568)
  • ผังเมืองรวมทั่วประเทศ — DPT ArcGIS
  • พ.ร.บ.ควบคุมอาคาร พ.ศ.2522 (แก้ไขถึงฉบับที่ 5)
  • กฎกระทรวง ฉบับที่ 33, 55 ฯลฯ
  • ข้อบัญญัติท้องถิ่น (กทม., เทศบาล)
  • พ.ร.บ.ผังเมือง พ.ศ.2562
  • ประกาศ EIA/EHIA

ข้อมูลตลาดและราคา

  • ราคาประเมินกรมที่ดิน
  • ราคาตลาดจาก DDProperty, Tooktee, Baania
  • ต้นทุนก่อสร้างต่อ ตร.ม. (SCG Price List)
  • อัตราดอกเบี้ยสินเชื่อโครงการ
  • ค่าธรรมเนียมโอน + ภาษี
  • Absorption rate ของพื้นที่
